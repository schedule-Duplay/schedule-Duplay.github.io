Gambaran Umum Modul

Modul ini membahas tentang Berpikir Komputasional dengan fokus pada:

  • Konsep struktur data array
  • Algoritma pengurutan standar (bubble sort, selection sort)
  • Analisis eksekusi algoritma (tracing)

Tujuan Pembelajaran

  1. Menerapkan struktur data array untuk menyimpan dan mengelola data
  2. Mensimulasikan algoritma pengurutan standar
  3. Menganalisis eksekusi algoritma pencarian atau pengurutan

Struktur Data Array

Apa itu Array?

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dalam satu variabel.

Bubble Sort

Asesmen Pembelajaran

Kuis Pra-Pembelajaran

Soal 1

Dalam pemrograman, jika Anda ingin menyimpan satu data, misalnya skor nilai ulangan Budi yaitu 90, Anda akan menyimpannya dalam sebuah "wadah" bernama yang disebut...

A. Array
B. Algoritma
C. Variabel
D. Tipe Data

Jawaban Benar: C. Variabel

Variabel adalah "kotak" dasar dalam memori yang diberi nama untuk menyimpan satu nilai.

Soal 2

Diberikan sekumpulan angka acak: [15, 8, 20, 11]. Apakah yang dimaksud dengan mengurutkan data tersebut secara ascending (menaik)?

A. Mencari angka terbesar, yaitu 20.
B. Menghitung jumlah semua angka.
E. Menyusun ulang angka menjadi [8, 11, 15, 20].

Jawaban Benar: E. Menyusun ulang angka menjadi [8, 11, 15, 20]

Ascending (menaik) berarti mengurutkan dari nilai terkecil ke nilai terbesar.
