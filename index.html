Konsep Dasar Array

Array adalah struktur data yang digunakan untuk menyimpan kumpulan data dengan tipe yang sama dalam satu variabel. Bayangkan array sebagai sekumpulan kotak berurutan yang masing-masing dapat menampung satu nilai.

// Deklarasi array dalam JavaScript
let scores = [85, 92, 78, 96, 88];

// Mengakses elemen array
console.log(scores[0]); // Output: 85
console.log(scores[2]); // Output: 78

// Mengubah nilai elemen array
scores[1] = 95; // Sekarang scores = [85, 95, 78, 96, 88]

// Menambah elemen baru
scores.push(90); // scores = [85, 95, 78, 96, 88, 90]

Kegiatan Interaktif: Simulasi Array

Gunakan kontrol di atas untuk menambah, menghapus, atau mengacak nilai array. Perhatikan bagaimana array memungkinkan kita menyimpan dan mengelola banyak data dalam satu struktur.

Pertanyaan: Jika kita memiliki array dengan nama "buah" yang berisi ["Apel", "Jeruk", "Mangga"], apa yang akan terjadi jika kita menjalankan perintah buah[1] = "Pisang"?

Array akan menjadi ["Pisang", "Jeruk", "Mangga"]
Array akan menjadi ["Apel", "Pisang", "Mangga"]
Array akan menjadi ["Apel", "Jeruk", "Pisang"]
Terjadi error karena indeks tidak valid

Penjelasan: Dalam pemrograman, indeks array dimulai dari 0. Jadi, buah[1] merujuk pada elemen kedua dalam array, yaitu "Jeruk". Menggantinya dengan "Pisang" akan menghasilkan array ["Apel", "Pisang", "Mangga"].

Algoritma Pengurutan: Bubble Sort

Bubble Sort adalah algoritma pengurutan sederhana yang berulang kali membandingkan elemen-elemen yang berdekatan dan menukarnya jika urutannya salah. Proses ini diulang hingga tidak ada lagi pertukaran yang diperlukan.

// Implementasi Bubble Sort dalam JavaScript
function bubbleSort(arr) {
    let n = arr.length;
    let swapped;
    
    do {
        swapped = false;
        for (let i = 0; i < n - 1; i++) {
            // Bandingkan elemen yang berdekatan
            if (arr[i] > arr[i + 1]) {
                // Tukar jika urutannya salah
                [arr[i], arr[i + 1]] = [arr[i + 1], arr[i]];
                swapped = true;
            }
        }
        n--;
    } while (swapped);
    
    return arr;
}

// Contoh penggunaan
let numbers = [64, 34, 25, 12, 22, 11, 90];
console.log(bubbleSort(numbers)); // Output: [11, 12, 22, 25, 34, 64, 90]

Status: Siap memulai pengurutan

Perbandingan: 0

Pertukaran: 0

Analisis Tracing Algoritma

Tracing adalah proses menelusuri eksekusi algoritma langkah demi langkah untuk memahami cara kerjanya dan menganalisis hasil akhirnya.

Contoh: Tracing Bubble Sort

Mari kita telusuri eksekusi Bubble Sort pada array: [10, 5, 12]

Langkah 1: Bandingkan elemen pada indeks 0 (10) dan indeks 1 (5)

10 > 5? Ya, jadi kita menukarnya.

Array sekarang: [5, 10, 12]

Langkah 2: Bandingkan elemen pada indeks 1 (10) dan indeks 2 (12)

10 > 12? Tidak, jadi tidak ada pertukaran.

Array tetap: [5, 10, 12]

Langkah 3: Satu putaran selesai. Periksa apakah masih ada pertukaran?

Dalam putaran ini terjadi 1 pertukaran, jadi kita lanjutkan.

Langkah 4: Bandingkan elemen pada indeks 0 (5) dan indeks 1 (10)

5 > 10? Tidak, jadi tidak ada pertukaran.

Array tetap: [5, 10, 12]

Langkah 5: Tidak ada pertukaran dalam putaran ini, algoritma selesai.

Array terurut: [5, 10, 12]
